# Pharmacy Images

Place pharmacy images in this directory.

## Example:

- himalayan-pharmacy.jpg
- [pharmacy-name].jpg

Images should be:

- Format: JPEG or PNG
- Recommended size: 800x600px or larger
- Aspect ratio: 4:3 or 16:9

## Usage

The images are referenced in the pharmacy profile data. For example:

```typescript
image: "/src/assets/images/pharmacies/himalayan-pharmacy.jpg";
```

Make sure to update the image path in your pharmacy data to match the actual image file name.
